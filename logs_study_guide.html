Exponential & Logarithmic Equations and Functions

A catch-up walkthrough: every method, formula, and question type from the three worksheets you missed — with worked examples, self-checks, and practice tools built in.

Read top to bottom once, then use the tools to test yourself.

§0 The one idea everything rests on

A logarithm is just an exponent in disguise. The whole chapter is built on one equivalence — read it in both directions until it feels automatic:

The master equivalence \[ a^{x}=y \quad\Longleftrightarrow\quad x=\log_{a} y \qquad (a>0,\ a\neq 1,\ y>0) \]

In words: \(\log_a y\) is the power you raise \(a\) to, to get \(y\). "Log base \(a\) of \(y\)" answers the question "\(a\) to the what \(=y\)?"

Because of this, the exponential function \(y=a^x\) and the logarithmic function \(y=\log_a x\) are inverses — each one undoes the other. That single fact powers:

  • Solving equations — if the unknown is stuck in an exponent, take a log; if it's stuck in a log, rewrite as an exponent.
  • Graphs — the log graph is the exponential graph reflected in the line \(y=x\).

§1 Logarithm fundamentals

1.1 Converting between index form and log form

Half the marks in this topic come from fluently switching between the two forms. Same statement, two costumes:

Two forms of the same fact \[ \underbrace{a^{x}=y}_{\text{index form}} \quad\Longleftrightarrow\quad \underbrace{\log_{a} y = x}_{\text{log form}} \]
Try it before you look
Worked example — index → log, then log → index

Convert \(2^{3}=8\) and \(3^{-2}=\tfrac19\) to log form. Then convert \(\log_{5}25=2\) and \(\log_{k}3.5=\pi\) to index form.

1\(2^{3}=8 \;\Rightarrow\; \log_{2}8 = 3\)
2\(3^{-2}=\tfrac19 \;\Rightarrow\; \log_{3}\tfrac19 = -2\)
3\(\log_{5}25 = 2 \;\Rightarrow\; 5^{2}=25\)
4\(\log_{k}3.5=\pi \;\Rightarrow\; k^{\pi}=3.5\)
Trick to never mix it upThe base of the log is the base of the power. The answer of the log is the exponent. Read log_a y = x as "a raised to x gives y."

1.2 When is a logarithm defined?

Three conditions — questions love to test them by giving you an "answer" that is secretly invalid.

Domain conditions for \(\log_a y\) \[ a>0,\qquad a\neq 1,\qquad y>0 \]

Quick intuition for each:

  • \(y>0\) — a positive base raised to any power stays positive, so you can never get a zero or negative output. Hence you can't feed a zero/negative into a log.
  • \(a\neq 1\) — since \(1^{x}=1\) for every \(x\), \(\log_1 y\) has no single value (it would be undefined).
  • \(a>0\) — negative or zero bases give undefined or imaginary results for non-integer powers.
This causes lost marksAfter solving a log equation you must check every solution against \(y>0\) for each log in the original equation. A value that makes any argument \(\le 0\) is rejected. (See §4.4.)

1.3 Special values you should recognise instantly

Memorise these four \[ \log_{a}1 = 0 \qquad \log_{a}a = 1 \qquad \log_{a}a^{\,n}=n \qquad a^{\log_{a}x}=x \]
  • \(\log_a 1=0\) because \(a^{0}=1\).
  • \(\log_a a=1\) because \(a^{1}=a\).
  • \(\log_a a^{n}=n\) because \(a^{n}=a^{n}\) — the log and the power cancel.
  • \(a^{\log_a x}=x\) — raising \(a\) to "the power that gives \(x\)" gives back \(x\).

1.4 Common logs (lg) and natural logs (ln)

Two special bases \[ \lg x \equiv \log_{10} x \qquad\qquad \ln x \equiv \log_{e} x \quad (e\approx 2.71828\ldots) \]

From the definition: \(\ \lg x = y \Leftrightarrow 10^{y}=x\ \) and \(\ \ln x = y \Leftrightarrow e^{y}=x.\)

These are the only two bases your calculator has buttons for — exactly why they matter for §5.

\[ \lg 10^{x}=x,\qquad \ln e^{x}=x,\qquad 10^{\lg x}=x,\qquad e^{\ln x}=x \]

§2 The laws of logarithms

2.1 The three laws

These let you break apart or combine logs. They only work when all logs share the same base.

Laws of logarithms (same base \(a\), with \(x,y>0\)) \[ \textbf{Product: }\ \log_a(xy)=\log_a x+\log_a y \] \[ \textbf{Quotient: }\ \log_a\!\Big(\tfrac{x}{y}\Big)=\log_a x-\log_a y \] \[ \textbf{Power: }\ \log_a\big(x^{\,n}\big)=n\log_a x \]
  • Expanding (one big log → several small ones): useful when a question says "express in terms of \(\lg 3\), \(\lg 5\)…".
  • Condensing (several logs → one log): useful before solving an equation, so you can strip the logs off.
Try it before you look
Worked example — express in terms of \(x=\log_p 3\) and \(y=\log_p 5\)

Express \(\log_p 75\) and \(\log_p\dfrac{27}{25}\) in terms of \(x\) and \(y\).

1\(75=3\times 5^{2}\), so \(\log_p 75=\log_p 3+\log_p 5^{2}=\log_p 3+2\log_p 5\).
2That gives \(\log_p 75 = x+2y\).
3\(\dfrac{27}{25}=\dfrac{3^{3}}{5^{2}}\), so \(\log_p\dfrac{27}{25}=3\log_p 3-2\log_p 5\).
4That gives \(\log_p\dfrac{27}{25}=3x-2y\).

2.2 Two more results worth knowing

\[ a^{\log_a x}=x \qquad\text{and}\qquad \text{if } \log_a x=\log_a y \text{ then } x=y \]

The second one ("logs cancel on both sides") is the engine behind the equation method in §4.1.

2.3 Misconceptions that wreck answers

These are NOT true — do not do them \[ \log_a(x+y)\;\neq\;\log_a x+\log_a y \] \[ \log_a(x-y)\;\neq\;\log_a x-\log_a y \] \[ \log_a(xy)\;\neq\;\log_a x \cdot \log_a y \] \[ \big(\log_a x\big)^{n}\;\neq\; \log_a\big(x^{\,n}\big)\ \ \text{(i.e. }(\log x)^2 \ne \log x^2) \] \[ \dfrac{\log_a x}{\log_a y}\;\neq\;\log_a\!\Big(\tfrac{x}{y}\Big) \]
The one to burn in\((\log x)^2\) and \(\log(x^2)\) are completely different. \((\log x)^2 = (\log x)(\log x)\); but \(\log(x^2)=2\log x\). This distinction is the whole point of §6.
Spot the mistake

Each statement is either a valid law or a common misconception. Decide True or False before revealing the answer.

2.4 Change of base

When two logs have different bases, convert them to a common base first. Pick base 10 or \(e\) if you need a calculator value.

Change-of-base formula \[ \log_a b=\frac{\log_c b}{\log_c a}\qquad(\text{any valid base }c) \]

Special case (swap): \(\ \log_a b=\dfrac{1}{\log_b a}\).

Example — evaluate to 3 s.f.

\(\log_{7}25=\dfrac{\lg 25}{\lg 7}=\dfrac{1.39794}{0.845098}=1.65\)

Example — no calculator (choose a clever base)

\(\log_{8}32=\dfrac{\log_2 32}{\log_2 8}=\dfrac{5}{3}=1\tfrac23\)

§3 Solving exponential equations — the two "no-log" tricks

Before logs, there are two situations where you don't need logs at all. Recognise them first.

Method A — Compare indices (make the bases equal)

When to useBoth sides can be written as powers of the same base. Then just equate the exponents.
Try it before you look
Worked example

Solve \( \dfrac{2^{x}}{4^{x-3}}\times 8^{3x+1}=0.25\).

1Write everything in base 2: \(4=2^{2}\), \(8=2^{3}\), \(0.25=\tfrac14=2^{-2}\).
2\(\dfrac{2^{x}}{2^{2(x-3)}}\times 2^{3(3x+1)}=2^{-2}\).
3Add/subtract indices: \(2^{\,x-2(x-3)+3(3x+1)}=2^{-2}\ \Rightarrow\ 2^{\,8x+9}=2^{-2}\).
4Equate indices: \(8x+9=-2 \Rightarrow x=-\tfrac{11}{8}\).

Method B — Substitution (a hidden quadratic)

When to useYou see a term and its square, e.g. \(4^{x}=(2^{x})^{2}\), or \(9^{x}\) with \(3^{x}\). Let \(u=\) the smaller power.
Try it before you look
Worked example

Solve \(4^{x+1}-\big(2^{x}\big)^{2}=12\).

1Rewrite in terms of \(2^x\): \(4^{x+1}=4\cdot 4^{x}=4(2^{x})^{2}\).
2Let \(u=2^{x}\): \(4u^{2}-u^{2}=12 \Rightarrow 3u^{2}=12 \Rightarrow u^{2}=4 \Rightarrow u=2\) (reject \(u=-2\), since \(2^x>0\)).
3Back-substitute: \(2^{x}=2 \Rightarrow x=1\).
Golden rule after substitution\(2^x\), \(e^x\), \(3^x\)… are always positive. So reject any negative value of \(u\) before back-substituting.

§4 Solving logarithmic equations

Three set-ups, then the all-important validity check.

4.1 Same base on both sides → compare arguments

\[ \log_a x=\log_a y \ \Longrightarrow\ x=y \]
Example

\(\log_3(2x+1)=\log_3 5 \ \Rightarrow\ 2x+1=5 \ \Rightarrow\ x=2\).

4.2 A single log = a constant → switch to exponential form

Example

\(\log_3(2x+1)=2\)

→ 2x + 1 = 3² → 2x + 1 = 9 → x = 4

4.3 Several logs → condense with the laws first

StrategyUse product/quotient/power laws to collapse the equation into one log on each side (case 4.1) or one log = constant (case 4.2). Then finish as above.
Try it before you look
Worked example (full, including the validity check)

Solve \(\log_{2}(x-3)+\log_{2}(x-4)=1\).

1Condense the left with the product law: \(\log_{2}\big[(x-3)(x-4)\big]=1\).
2Switch to exponential form: \((x-3)(x-4)=2^{1}=2\).
3Expand and solve the quadratic: \(x^{2}-7x+12=2 \Rightarrow x^{2}-7x+10=0 \Rightarrow (x-5)(x-2)=0\).
4Candidates: \(x=5\) or \(x=2\).
5Validity check — need \(x-3>0\) and \(x-4>0\), i.e. \(x>4\).
6\(x=5\) ✓ works; \(x=2\) ✗ makes both logs undefined → reject. Final answer: \(x=5\).

4.4 Always check validity — the step examiners bait you on

Non-negotiable last step Substitute every solution back into the original equation. If any argument of any log becomes \(\le 0\), that solution is rejected. Solving the algebra correctly but keeping an invalid root still loses the mark.

4.5 When the unknown is the base itself

When to useAll the logs share the same base — even if that base is the unknown \(x\) — so the product/quotient laws still apply directly. Condense first, exactly as before.
Try it before you look
Worked example

Solve \(\log_x 2+\log_x 4+\log_x 8=6\).

1All three logs share the same base \(x\), so the product law still applies: \(\log_x(2\times4\times8)=6\).
2\(\log_x 64=6\).
3Switch to exponential form: \(x^{6}=64\).
4\(x=64^{1/6}=2^{6/6}=2\). Check: \(x>0,\ x\neq1\) — valid.
Extra validity condition hereSince \(x\) is the base of the log, you also need \(x>0\) and \(x\neq1\) — check this alongside the usual argument-positive condition.

§5 Exponential equations that do need logs

When the bases can't be matched (e.g. \(3^{x}=5\)), take \(\lg\) or \(\ln\) of both sides and use the power law to bring the unknown down from the exponent.

Core move \[ a^{x}=b \ \Longrightarrow\ \lg a^{x}=\lg b \ \Longrightarrow\ x\lg a=\lg b \ \Longrightarrow\ x=\frac{\lg b}{\lg a} \]

Use ln instead when the base is \(e\) (it cancels neatly). Answers here are usually asked to 3 s.f.

Example 2 — base \(e\), use ln

Solve \(e^{2x}=7\) (3 s.f.).

ln both sides → 2x = ln7 → x = ln7 / 2 = 1.94591 / 2 = 0.973
Try it before you look
Worked example — unknown in two different exponents

Solve \(2^{\,x+1}=3^{\,2x-1}\) (3 s.f.).

1Take lg both sides: \((x+1)\lg 2=(2x-1)\lg 3\).
2Expand: \(x\lg 2+\lg 2=2x\lg 3-\lg 3\).
3Collect \(x\)-terms on one side: \(x\lg 2-2x\lg 3=-\lg 3-\lg 2\).
4Factor \(x\): \(x(\lg 2-2\lg 3)=-(\lg 2+\lg 3)\).
5\(x=\dfrac{-(\lg 2+\lg 3)}{\lg 2-2\lg 3}=\dfrac{-\lg 6}{\lg 2-\lg 9}=\dfrac{-0.77815}{-0.65321}=1.19\).
Pattern for "\(x\) in two exponents"lg both sides → expand brackets → gather all \(x\)-terms one side, constants the other → factor out \(x\) → divide. It's always these five moves.

§6 Quadratic equations involving logs

These hide a quadratic inside a substitution. The tell-tale sign is a \((\log x)^2\) term (a squared log), often alongside a plain \(\log x\).

MethodLet \(u=\log_a x\). Rewrite the equation as a quadratic in \(u\), solve, then back-substitute \(u=\log_a x\) to find \(x\).
Try it before you look
Worked example

Solve \((\lg x)^{2}-3\lg x+2=0\).

1Let \(u=\lg x\): \(u^{2}-3u+2=0\).
2Factor: \((u-1)(u-2)=0 \Rightarrow u=1\) or \(u=2\).
3Back-substitute: \(\lg x=1\Rightarrow x=10\); \(\ \lg x=2\Rightarrow x=100\).
Example 2 — needs change of base first

Solve \(\log_{3}x = 16\,\log_{x}3\).

  1. Rewrite the awkward term with the swap rule: \(\log_{x}3=\dfrac{1}{\log_{3}x}\).
  2. Let \(u=\log_{3}x\): \(u=\dfrac{16}{u}\ \Rightarrow\ u^{2}=16\ \Rightarrow\ u=\pm 4\).
  3. Back-substitute: \(\log_3 x=4\Rightarrow x=3^{4}=81\); \(\ \log_3 x=-4\Rightarrow x=3^{-4}=\tfrac{1}{81}\).
Read the notation carefully first\((\log x)^2 = u^2\), but \(\log x^{2}=2\log x = 2u\). Mis-reading which one you have changes the whole problem.

§7 Simultaneous equations involving logs

Treat the logs (or the whole log expressions) as your two unknowns, solve the pair like a normal simultaneous system, then unwind back to \(x\) and \(y\). Same-base condensing (§2) usually clears the way.

Try it before you look
Worked example

Solve \(\ \log_{2}x+\log_{2}y=5\ \) and \(\ \log_{2}x-\log_{2}y=1\).

1Let \(P=\log_2 x\), \(Q=\log_2 y\): the system is \(P+Q=5,\ P-Q=1\).
2Add the equations: \(2P=6\Rightarrow P=3\). Subtract: \(2Q=4\Rightarrow Q=2\).
3Unwind: \(\log_2 x=3\Rightarrow x=2^{3}=8\); \(\ \log_2 y=2\Rightarrow y=2^{2}=4\).
4Check: \(\log_2 8+\log_2 4=3+2=5\) ✓, and \(3-2=1\) ✓.
Also commonOne equation is exponential (\(2^{x}\div 4^{y}=8\) → compare indices → a linear relation in \(x,y\)) and the other is logarithmic (condense → another relation). Solve the two linear-ish relations together.

7.2 Expressing a third combination in terms of p and q

When to useYou're given two log equations in \(x\) and \(y\), and asked for a different combination of \(x,y\) — not to solve for \(x\) and \(y\) themselves. Expand everything in terms of \(\log x\) and \(\log y\), solve that pair, then substitute into the target.
Try it before you look
Worked example

Given \(\log_{7}(x^{2}y)=p\) and \(\log_{7}\!\Big(\dfrac{x}{y^{2}}\Big)=q\), express \(\log_{7}(x^{3}y)\) in terms of \(p\) and \(q\).

1Expand both given equations: \(2\log_7 x+\log_7 y=p\) and \(\log_7 x-2\log_7 y=q\).
2Let \(X=\log_7 x\), \(Y=\log_7 y\): the system is \(2X+Y=p\), \(X-2Y=q\).
3Solve simultaneously: from the second equation \(X=q+2Y\). Substitute into the first: \(2(q+2Y)+Y=p \Rightarrow 5Y=p-2q \Rightarrow Y=\dfrac{p-2q}{5}\), and \(X=\dfrac{2p+q}{5}\).
4Target: \(\log_7(x^3y)=3X+Y=3\Big(\dfrac{2p+q}{5}\Big)+\dfrac{p-2q}{5}=\dfrac{6p+3q+p-2q}{5}=\dfrac{7p+q}{5}\).
PatternExpand → substitute \(X=\log x,\ Y=\log y\) → solve the linear system for \(X,Y\) → substitute into the target expression (also expanded in terms of \(X,Y\)).

§8 Applications (growth, decay, models)

Real-world questions dress the same maths in a story: populations, radioactive decay, cooling, depreciation, interest. They almost always use one of these forms:

Standard model shapes \[ y=A e^{kt}\quad(\text{continuous growth if }k>0,\ \text{decay if }k<0) \] \[ P=P_0(r)^{n}\quad(\text{e.g. }P_0(1.03)^n\text{ for }3\%\text{ per year}) \] \[ y=\frac{L}{1+Ce^{-kt}}\quad(\text{logistic / limited growth}) \]
General method
  1. Find the constants first. Substitute the "initial" data (often \(t=0\)) to get \(A\) or \(P_0\); substitute a second data point to find \(k\) (take a log to solve for it).
  2. To find a value at a given time: substitute \(t\) and evaluate.
  3. To find a time for a given value: substitute \(y\), then take \(\ln\)/\(\lg\) to solve for \(t\).
Try it before you look
Worked example — radioactive decay

A substance decays by \(y=Ae^{-0.2t}\) (mg, \(t\) in years). At the start (\(t=0\)) there is \(500\) mg. (a) Find the amount after 5 years. (b) Find when it is halved.

1Find \(A\): at \(t=0\), \(500=Ae^{0}\Rightarrow A=500\).
2(a) \(y=500e^{-0.2(5)}=500e^{-1}=500(0.3679)=184\) mg.
3(b) Set \(y=250\): \(250=500e^{-0.2t}\Rightarrow 0.5=e^{-0.2t}\).
4Take ln: \(\ln 0.5=-0.2t \Rightarrow t=\dfrac{\ln 2}{0.2}=\dfrac{0.6931}{0.2}=3.47\) years.
Example — exponential population growth

A population follows \(P=P_0e^{kt}\), growing from \(50{,}000\) (at \(t=0\)) to \(250{,}000\) after \(10\) years.

  1. \(P_0=50{,}000\). At \(t=10\): \(250000=50000e^{10k}\Rightarrow 5=e^{10k}\).
  2. \(10k=\ln 5\Rightarrow k=\dfrac{\ln 5}{10}=0.1609\).
  3. Population 2 years later (\(t=12\)): \(P=50000\,e^{0.1609(12)}=50000\,e^{1.931}\approx 345{,}000\).
Real-life contexts to expectpH \(=-\lg[\text{H}^+]\); decibels \(d=10\lg(I/I_0)\); the Richter scale (each point = ×10 intensity); Newton's Law of Cooling; compound-interest depreciation. All are just the models above with a story attached.
Try it before you look
Worked example — compound growth, solving for the exponent

A collectible's value follows \(P=53.50(1.03)^{n}\) after \(n\) years. (a) Find \(P\) when \(n=17\). (b) Find \(n\) when \(P=\$110\) (to the nearest month).

1(a) \(P=53.50(1.03)^{17}=53.50(1.6528)=\$88.43\) (2 d.p.).
2(b) Set \(P=110\): \(110=53.50(1.03)^{n}\ \Rightarrow\ (1.03)^{n}=\dfrac{110}{53.50}=2.0561\).
3Take lg both sides: \(n\lg 1.03=\lg 2.0561\ \Rightarrow\ n=\dfrac{\lg 2.0561}{\lg 1.03}=24.45\) years.
4\(0.45\) of a year \(\approx 5.4\) months, so \(n\approx 24\) years \(5\) months.

§9 Graphs of exponential & logarithmic functions

9.1 Exponential graphs \(y=ka^{x}\)

Key features of \(y=a^{x}\) (with \(a>0,\ a\neq1\))
  • y-intercept always \(1\) (since \(a^{0}=1\)).
  • Horizontal asymptote \(y=0\) (the \(x\)-axis) — the curve never touches it.
  • \(y>0\) for all \(x\) — the graph sits entirely above the \(x\)-axis.
  • Increasing if \(a>1\); decreasing if \(0 Bigger \(a\) (\(>1\)) → steeper.
Effect of \(k\) in \(y=ka^{x}\)

\(k\) sets the y-intercept (curve passes through \((0,k)\)). If \(k<0\) the whole graph flips below the \(x\)-axis.

  • \(y=a^{-x}\) is \(y=a^{x}\) reflected in the \(y\)-axis (and equals \((1/a)^{x}\)).
  • \(y=-a^{x}\) is \(y=a^{x}\) reflected in the \(x\)-axis.

9.2 Transformations

FormWhat changesy-interceptAsymptote
\(y=a^{bx+c}\)horizontal stretch/shift inside exponent\(a^{c}\) (put \(x=0\))\(y=0\)
\(y=e^{2x}-1\)whole graph shifts down 1\(1-1=0\)\(y=-1\)
\(y=e^{2x}+2\)whole graph shifts up 2\(1+2=3\)\(y=2\)
RuleAdding/subtracting a constant outside the power (\(e^{2x}\pm k\)) shifts the curve and its horizontal asymptote vertically by that amount.

9.3 Logarithmic graphs — the mirror image

Since \(y=\log_a x\) is the inverse of \(y=a^{x}\), its graph is the exponential graph reflected in the line \(y=x\).

Key features of \(y=\log_a x\) (with \(a>1\))
  • x-intercept always \(1\) (since \(\log_a 1=0\)).
  • Vertical asymptote \(x=0\) (the \(y\)-axis).
  • Domain \(x>0\) only — the graph exists only to the right of the \(y\)-axis.
  • As \(x\to 0^{+},\ y\to-\infty\). Increasing for \(a>1\), decreasing for \(0

9.4 Sketching \(y=\log_a(bx+c)\) — the reliable recipe

The three-step sketch
  1. Vertical asymptote: set the inside \(=0\) → \(bx+c=0\) → \(x=-\dfrac{c}{b}\). The curve hugs this line.
  2. x-intercept: set \(y=0\), so the inside \(=1\) → \(bx+c=1\) → \(x=\dfrac{1-c}{b}\).
  3. y-intercept: set \(x=0\) → \(y=\log_a c\) (only exists if \(c>0\)).

Shape: for base \(a>1\), the curve increases if \(b>0\) and decreases if \(b<0\). Domain: wherever \(bx+c>0\).

Try it before you look
Fully worked: \(y=\ln(x-1)\)
1Asymptote: \(x-1=0\Rightarrow x=1\). Draw a dashed vertical line there.
2\(x\)-intercept: \(x-1=1\Rightarrow x=2\). Mark \((2,0)\).
3Shape: base \(e>1\) and \(b=1>0\) → increasing. Curve rises from \(-\infty\) just right of \(x=1\), through \((2,0)\), climbing slowly.
Sketch checklist (mark-earning parts)Label the dashed asymptote with its equation, the x-intercept, the y-intercept (if it exists), and give the curve the correct increasing/decreasing shape approaching the asymptote.

Common-mistake radar

  • Never write \(\log(x+y)=\log x+\log y\). The laws are for products/quotients/powers only.
  • Keep \((\log x)^2\) and \(\log x^2\) apart — they are different.
  • After a substitution, reject negative values of \(2^x,\,e^x\) etc. (always positive).
  • After solving any log equation, check each root keeps every argument \(>0\); reject the rest.
  • For sketches, label the asymptote equation and both intercepts — that's where the marks are.
  • Read the required accuracy: most "solve" questions want 3 significant figures.
  • Match the base to the tool: base \(e\) → use \(\ln\); base \(10\) → use \(\lg\).

Everything above was grouped by method, which is the easiest way to learn a technique but the easiest way to fool yourself about being ready.
